前言

1978年，中国开始对内改革，对外开放

随后中国涌入了大量的外资外企

随之而来的是外国人

如果说一开始进来的外国人还只是很小数量，且大多聚集在北京上海这些一线城市，接触到的人很少，能造成的影响微乎其微

那么在改革开放的四十二年后，gdp增加了80倍还要多的今天，越来越强大的中国所吸引的越来越多的外国人所造成的影响就不容小觑了

当越来越多的人注意到在华外国人这个群体后，一些差异对待，外国人违法事件往往会得到许多人的关注，而从舆论来看。。。。我们的基层部门往往做的还不够好

这也是难免的，因为一些历史原因，这种话题对国人的神经来说本来就比较敏感，大部分地区也缺乏处理这一类事务的经验，想大事化小小事化了往往却事与愿违

于是想记录一下，每一起事情的处理结果，最后希望能看到我们处理的进步吧

1840年，是一个庚子年，那一年中国被迫打开了国门

2020年，也是一个庚子年，正好要记录的事情也和外国人有关，所以起了个名字叫庚子年备忘录，只要不懒得话大概会一直更新吧。。（说的好像这个博客除了你还会有人看一样qrz）嗯。。介绍完了，下面是正文

“为了684个老外的安康”文章事件

时间：3月26日

国籍：大量各国外籍人员

人种：未知

冲突原因：差异性对待

事情经过：3月26日，《南京日报》刊文《为了684个“老外”的安康》：有的老外一次只买4片面包，确保每天吃新鲜的，我们就每天送上门；老外要喝桶装纯净水，一次性购买了4大桶，我们就帮他一桶一桶从小区门口扛到楼上；有一位外国友人买了大件物品，没有电梯，我们硬是派了两个人抬上4楼、送进家里；有个小年轻酷爱淘宝，我们有一天帮他送了20多趟快递……

累计684名“老外”享受到了“包保服务”，一户一个工作组对口服务，让仙林外防输入的工作“忙而不乱”。这两天，工作组成员们还增添了新任务，那就是帮助外籍人士采购口罩等防疫物资，然后快递回老家，尤其是疫情严重的意大利、韩国等地区。

澳籍华人在疫情期间坚持要跑步且不带口罩事件

时间：3月15日

国籍：澳大利亚籍华人

人种：黄种人

地点：北京朝阳区

冲突原因：此事处理的较好，舆论倾向于肯定，但值得一提的是，在此事发生时间附近，有一华人女子（中国籍）在隔离区要喝矿泉水受到舆论攻击，因为是中国籍，未列入此备忘录

事情经过：3月15日15时许，朝阳公安分局接某小区卫生防疫工作人员报警称：一女子拒不配合社区防疫工作。接报警后，呼家楼派出所民警立即赶赴现场开展工作。经核查，梁某妍，女，47岁，澳大利亚籍，就职于拜耳医药保健有限公司，3月14日由首都机场入境进京，工作居留许可有效期至2020年9月5日。15日下午，本应在租住地居家观察的梁某妍，未戴口罩在小区内跑步，社区卫生防疫工作人员发现后进行劝阻，但该人情绪激动，拒不配合。民警到场后，对梁某妍进行了批评教育，要求其严格遵守疫情防控相关规定，梁某妍表示服从管理，未再外出。

最新处理结果：该女子被所在企业拜耳开除，3月18日北京市公安局出入境管理局依据《中华人民共和国出境入境管理法》第六十七条等规定，决定依法注销梁某妍工作类居留许可、限期离境。

青岛崂山区志愿者一对一服务外国人事件

时间：3月份

国籍：未知

人种：未知

地点：山东省青岛市崂山区

冲突原因：差异性对待

事情经过：网络上青岛志愿者一对一服务外籍隔离人员走红，视频中志愿者为外国人买猫粮，送快递，且为一对一服务，之后崂山区金家岭街道疫情防控指挥部工作人员成这是社区提供的人性化服务

最新处理结果：金家岭街道办事处工作人员对该事件作出回应，“入境人员实行集中隔离的政策是在3月20日晚下达的，从这之后，才开始入境人员集中隔离，在此政策下达之前的返青人员都是可以居家隔离的。”防控指挥部工作人员说道，因为刚开始没有集中隔离的酒店，所以实行居家隔离。当问及视频中“一对一志愿者服务”时，工作人员表示，这可能是社区提供的一种“人性化”服务。“一对一服务”具体要看其所居住的社区，政府并没有出台明确规定要求志愿者对外国人进行一对一服务
