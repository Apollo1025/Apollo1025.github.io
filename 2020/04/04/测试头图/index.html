2020年四月四日，自己的博客终于开始正式营业啦

最初萌生这个念头大概是在高三，原因是看到许多大牛都有自己的博客，向大牛们学习，当时就想高考完一定也开设一个自己的博客然后好好学习，绝！对！不！打！游！戏！

但是呢，王境泽虽然会迟到但绝不会缺席，高考完坚持了几个小时的计划表，就去打游戏了，之后便是颓废了一个假期，高考前定下的高考后要完成的事都抛在脑后了

稀里糊涂的上了大学，学到的东西越来越多，咸鱼如我也觉得不记点笔记好像不太行，但是电脑里文件乱七八糟，日常的记完笔记就扔到一个随便取名的文件夹，之后就留在那里让他生灰

​ 但这时候还没想起来博客这种形式

真正的想起来的原因还是因为自己在各种网站被封号删评

以及发现这个网络好像对一些事情的遗忘速度特别快，今天的热点因为明天新的热点而无人关注，一起又一起的敏感事件，最终都不知道结果。

于是，二者相结合，决定得想办法整个个人网站

记录学到的技术并尝试保持个人思想与言论独立——至少有形式上的独立，以及对如今逐趋完整的局域网保持谨慎。在一个你发在公众号、微博、豆瓣、知乎的言论会被莫名其妙地消失的时代，一个独立博客仍会让你有免于恐惧的自由。

[^自由引导人民]:

比较之后选择了hexo，加上githubpage，nodejs，因为hexo搭建成本是真的低= =,并且用的人比较多，遇到的各种问题基本都能靠百度+谷歌找到解决办法

主题选择了archer（fate？），没多想，看到作者头像是misaka就选了，炮姐赛高！！

域名选择了flipped1022.cn，flipped，没错，就是那个名叫怦然心动的电影，猛男必看

缺点就是，写博客是真的麻烦，需要在vim里用markdown语言写，后来想起来学长推荐的typora，简单了不少，界面终于不那么反人类了哈哈哈哈哈哈

最后，欢迎来到我的博客

我们记录，我们铭记，我们探索

缺失模块。
1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save
3、在根目录_config.yml里添加配置： 
jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: true
    tags: true
